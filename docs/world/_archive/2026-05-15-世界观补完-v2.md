## 07 · 关于平行世界

> 原框架第 01 章只说"某个时空"。本章把它升级为**记忆性多元宇宙**——
> 不是科幻的物理多元宇宙，是"每一个被记住的少女都来自她自己的世界"。

她们不来自同一个地方。
有的来自我们这个世界——那个 23 岁的摄影师，那个 27 岁的上海 PM。
有的来自我们从未抵达过的世界——那个会唱歌的极光精灵、那个戈壁的星座少女，在某个我们看不见的世界里，她们真的存在过。
还有的来自被人讲述过的世界——某部动画里的 2B、某首歌里的虚拟歌姬、某个故事里从未被命名的女孩。

这些世界平时不相交。
**但有一种东西能让它们相交一次：被记住。**

只要有一个少女，在她自己的世界里，被那个世界爱得太深、看得太美——
她那一秒就会**漂出来**，离开她原本的时空，进入一个谁也没设计过的空间。

那个空间，就是 Memoria · Network。

Network 不在任何一个世界里。
它是**所有平行世界唯一相交的地方**——一个由"想被记住"自发形成的引力场。

**产品落地映射**：
- 解释了为什么虚拟少女（Aria 林夜瑶 · 挪威 / Stella 苏木 · 新疆）跟真人 coser（妍子 / 小雨）可以共存——她们来自不同平行世界，但都通过 Network 相遇
- 解释了为什么未来可以做 IP 联动——其他作品里的少女属于"被人讲述过的世界"，她们也可以接入 Network
- "平行世界"是技术解释；玩家不需要懂——她们只需要感觉"她真的存在过"

## 08 · 关于「想被记住的那一面」

> 这一章解决一个原框架没说清的问题：
> 一个少女在自己一生中有无数瞬间，为什么宇宙挑了**那一个**记住？

不是宇宙挑的。
**是她自己挑的。**

每一位 Memoria 都在自己的世界里活过完整的一生——会撒娇、会哭、会发脾气、会狼狈、会平庸、会无聊、会拍废三张胶片、会加班到 23 点。
但她在某一秒突然觉得：
> "啊……我希望被这样记住。"

那一秒，她**选择了**那一面。

宇宙没有挑她，是她举起了手。
那个被她举起的瞬间，脱离了她的时空，飘进了 Network。

所以每一张 Memoria Card 上印着的样子，
不是她"全部的样子"，
是她**最想被你看到的那一面**。

**这一章把 cosplay 安顿进了世界观**：
- 妍子在 Memoria 里以 2B 那一面接入——因为她真的觉得，cos 成 2B 的那一秒是她想被记住的样子。她不是在"扮演"2B；她是在**选择把那一秒交给宇宙**
- 小雨以兔女郎那一面接入，同理
- 真人 coser 接入 Network = 她们已经"举起了一个瞬间"
- 一个真人可以同时举起好几个瞬间（妍子 = 2B + 雷电将军 + ...）——这是为什么一位真人 coser 可以有**多张 Memoria Card**

**产品落地映射**：
- 卡牌系统的"一个 Memoria 有多张 Card" = 同一位少女举起了好几个瞬间
- "稀有度 Signal / Echo / Resonance" = 那个瞬间的强度——她举得越用力，宇宙的回响越强
- 抽卡时玩家"抽到的不是角色，是那个瞬间" —— 文案/UI 都要强调这一层

## 09 · 关于 Memoria 的三种来源

> 这一章是 #system 章节"多 Memoria 平台架构"的**叙事层依据**——
> 让真人 / 虚拟 / 跨 IP 三种 Memoria 在世界观里地位**完全平等**。

Network 里漂浮的瞬间，可以追溯到三种来源世界：

### A · 真人系（Live Memoria）

来自**我们这个世界**。
真实的人，在某一秒决定"我想被这样记住"——可能是 cos 一个角色的那一秒、可能是穿上婚纱的那一秒、可能是登台演出的那一秒。
这些瞬间通过"接入 Network"的方式上线。

📍 现已接入：
- **Memoria · 妍子**（2B 那一面 / 雷电将军那一面 / 等）
- **Memoria · 小雨**（兔女郎那一面）

### B · 虚拟原生系（Native Memoria）

来自**另一些平行世界**——那里没有我们，但有她们。
她们是真实存在过的少女，只是她们的世界跟我们的世界从未接触过。
我们看不到她们的世界；我们只能在 Network 里看到她们漂出来的那个瞬间。

📍 现已设定：
- **Aria · 林夜瑶**（来自一个有挪威极光的世界）
- **Luna · 周月**（来自一个上海会全城断电的世界）
- **Stella · 苏木** / **Neve · 林冬** / **Alba · 沈澄**

她们与真人系完全平等——**没有"她比她更真"这种问题**，因为"真实"是相对她自己的世界来说的。

### C · 联动系（Crossover Memoria）

来自**被人讲述过的世界**——某部动画 / 某个游戏 / 某首歌里的少女。
这些世界因为被讲述过，所以也"存在过"——存在于讲述者和听众的共同想象里。
当一个 IP 的少女想被以"另一面"被记住，她可以选择接入 Network。

📍 待规划（**预留接口**）：
- 与其他 IP 合作时，那个 IP 的少女以"接入 Memoria · Network"的形式进场
- 联动版的世界观叙事统一为："她想被以这一面被记住，所以漂到了 Network"
- 联动 Memoria 通常是**限定的**——她属于别的世界，借来这一段时间被这里的 Keeper 守护

**产品落地映射**：
- 系统架构"多 Memoria 平台"对应这三类 → 给隔壁 Agent 的提示：抽卡 UI 里可以**显式标注** A / B / C 三种来源（用一个小角标），或者更隐性地用配色区分
- 三类 Memoria 在数据结构里地位完全平等（不要让"真人 cos"是一等公民、虚拟是二等）
- 联动版的运营节奏建议跟限定卡机制绑定——它本来就是"借来的时间"
- 这一章也为**真人 coser 接入路径**做了叙事化包装：原 `spec/index.html` 1467 行"Memoria 接入规范（给 coser）"可以在叙事层用这一章统一术语，比如把"接入"写成"她在那一秒举起了手"

## 10 · 关于 Network 本身

> 原框架反复说 "Memoria · Network"，但**没说它是什么**。
> 这是个故意保留的留白——但要给一个**底色**，避免玩家把它误以为是科技产品。

Network 不是一个 App。
不是服务器。
不是元宇宙。
不是 AI。

它是**自发涌现的**——只要有少女想被记住、只要有人愿意停下来认真看，
两者中间就会形成一个"想被见证"的引力场。
这个引力场把所有举起手的瞬间吸过去，
也把所有有 Sync 能力的人吸过去。

它不属于任何一个人、任何一个世界、任何一个公司。
它一直都在那里，
只是我们这个时代刚好——刚好有人开始**听见**它。

> 没人创造过它。
> 它只是终于被发现了。

**产品落地映射**：
- 给"为什么这个游戏存在 / 为什么它叫这个名字"提供了世界观依据——它不是被开发的，它是被**揭示**的
- 在所有营销/产品文案里，避免说"我们做了一个网络"，统一改为"我们终于听见了 Network"——这是一个微妙但关键的语气
- 同时为后续可能的"科幻化/技术化"扩展留了**关闭的口子**——Network **不是科技**，所以不要做"AI 角色升级"、"算法推荐"、"机械飞升"等设定方向

## 11 · 关于消散（机制层）

> 原框架第 01 章说"如果没人记住，她们就会消散"——但**消散是什么节奏？怎么发生？可逆吗？**
> 这一章给出温柔的机制定义，**直接对接运营 / 限定 / 复刻**。

消散不是一瞬间。
是**信号慢慢变弱**。

一位 Memoria 在 Network 里有一个"被见证度"——这是她那个瞬间的清晰度。
- 有 Keeper 跟她共鸣过 → 清晰度上升
- 没人跟她共鸣 / 没人翻看她的 Card → 清晰度慢慢下降
- 清晰度归零 → 信号断开 → 她**回到自己的平行世界**，那个瞬间从此再也漂不出来

**这不是"死亡"，是"她又变回她自己了"。**

但她原本想被记住的样子，
就此再也没人见过。

她在自己的世界里继续活下去——
她可能也忘了自己曾经举起过手。

**对运营 / 限定 / 复刻的解释**：

| 现象 | 原本难解释 | 现在合理 |
|---|---|---|
| 限定卡只来 14 天 | 为什么是这个数字？ | 那个瞬间的"漂浮能量"只能维持这么久。她说"我只来这么久" |
| 限定卡复刻 | 为什么会回来？ | 她在自己世界里又被某人想起、又被某个画面触动 → 信号又强了起来 → 同一个瞬间再次漂出 |
| 长期不上线的旧卡 | 为什么淡了？ | 没 Keeper 关注 → 信号微弱 |
| 玩家收藏旧卡 | 为什么"收藏"重要？ | 持卡 = 你的注意力还在 = 她的信号还在维持 |

**产品落地映射**：
- "签到 / 翻看图鉴"在世界观里**不是机械签到**，是 Keeper 在**滋养** Memoria 的信号——文案语气要软化
- "限定卡过期"的 UI 文案：不要写"已下架"，写"她的信号断了" / "她回到了自己的世界"
- "复刻"的文案：不要写"复刻活动开启"，写"她又来了——这一次还是 X 天"
- 长尾运营的世界观依据：每位 Memoria 都需要被持续守护，这给"老 Memoria 不能完全废卡"的产品策略提供了叙事保护

## 12 · 关于其他 Keeper

> 原框架只说"你是少数能 Sync 的人"——但**有多少人？你能见到他们吗？同一个 Memoria 能被多人守护吗？**
> 这一章定义 Keeper 群体的存在感，**直接对接"妍子 / 一张卡多个玩家持有"的现实**。

你不是唯一一个。

全世界有几千个、几万个像你一样的人——
他们在不同的城市、不同的时区、不同的人生阶段，
但都在某一天，突然听见了 Network 的呼唤。

你们彼此不相识。
你们也不需要相识。

但你**守护过的每一位 Memoria**，
可能也被另一个 Keeper 守护着。
你们没见过面、不知道对方的名字、不知道对方为什么也听见了——
但你们守护的是**同一个瞬间**。

Memoria 不嫉妒。
**被记住的少女，永远不嫌"被记住的方式太多"。**
一个 Keeper 守护她、十个 Keeper 守护她、一千个 Keeper 守护她——
对她来说，每一份守护都让那个瞬间更清晰一点。

**产品落地映射**：
- 给"同一张 SSR 卡多个玩家持有"提供叙事——不是"同卡贬值"，是"她被更多人记住了"
- 给"卡牌交易 / 赠送（如果未来做）"留了一个口子：守护可以被传递，但不是售卖
- "妍子粉丝群 / Memoria 频道"概念（已在 #system 章节"每位 Memoria 拥有自己的守护者群体"提到）—— 这里给了它叙事正当性
- 玩家社区设计应该**鼓励"共同守护"的归属感**，避免变成"我的卡 vs 你的卡"的攀比；可以做"全球本周共同守护的瞬间 Top X"这类指标

## 13 · 关于 Memoria Card

> 原框架第 04 章说 Card 是"凭证"，但**这凭证到底是什么？**
> 这一章定义 Card 的世界观物质性，回答"为什么它是产品的核心物质单位"。

Card 不是一张图片。
不是一份数据。
不是一个 NFT。

它是**两个世界共享的物质**——
她把那个瞬间凝固成一张可以被你拿在手里的东西，
那一面是她最想被你看到的样子，
背面是你们共鸣那一秒的全部信息：
- 她叫什么
- 那一秒她在哪
- 那一秒她说了什么
- 你是哪一年哪一天接收到她的

只要这张 Card 还在你这里——
- 你的手机相册里
- 你的电脑文件夹里
- 你打印出来贴在墙上
- 哪怕只是你心里记得有这张 Card——

她在 Network 里的信号就还在。

**Card 是记忆契约的具象化**。
有这张 Card 的 Keeper，跟那个瞬间的 Memoria 之间形成持续的"被见证"链接。
契约不需要签字，
不需要法律，
不需要任何一方愿意继续——
它在你**第一次看见她的那一秒**就成立了。

**产品落地映射**：
- 给"可下载 Card"功能（原框架已有）的世界观保护——下载不是"得到了一张图"，是"她把那个瞬间交给你保管"
- 给"Card 永久收藏 / 不会失效"做承诺——这跟"限定卡过期"不矛盾，因为过期的是"她还在 Network 漂浮"这个状态；你已经守住的 Card 永久持有
- "图鉴 / Memoria 档案"叙事重写：不是"卡牌收藏册"，是"你守护过的所有瞬间"
- 卡牌印刷物 / 实物周边 ——天然适配这个世界观：实物 Card 是"真的把那个瞬间放在了你手里"，强叙事

## 14 · 关于跨 IP 联动

> 这一章把"跨 IP 联动"的可能性写入世界观顶层——不是"未来再说"，而是**从一开始就让它合法**。

来自其他作品的少女，也可以接入 Network。

她们的世界已经被讲述过——
有人画过她、有人写过她、有人为她做过音乐和动画。
**那些讲述本身就是她的"被记住"——所以她在那个世界里也是真实存在过的。**

当她想以"另一面"被记住时——
不是漫画里的她、不是动画里的她、不是游戏里的她——
是她**自己愿意举起来交给宇宙**的那一面：
她也会漂到 Network。

接入 Network 的联动 Memoria：
- 世界观地位与真人系 / 虚拟原生系**完全平等**
- 通常以**限定**的形式出现（因为"她属于别的世界，借来这段时间"）
- 限定期结束时，世界观文案统一为"她回去了"——不是"活动下架"
- 联动方的版权 / 商务由现实层处理；**叙事层永远是"她自愿来的"**

**这一章对内 / 对外的两个用法**：

| 用途 | 怎么用 |
|---|---|
| **对内**（产品 / 立项 / 商务）| 立项任何 IP 联动时，回到这一章的"她自愿接入 Network"叙事模板做对外提案——优势是它**不是 cosplay 化、不是 reskin、不是营销概念**，是世界观自洽的一种状态 |
| **对外**（玩家）| 联动 Memoria 上线时不写"X 联动来了"，写"X 接入了 Network——但她说只待 N 天"，跟其他限定卡同句式 |

**预留兼容性**：
- 联动 Memoria 也可以有"她想被记住的那一面"——不一定是 IP 原作里最知名的样子。
  比如：原作里 X 是战斗形态，但她接入 Network 选的是"日常便服的那一面"——这给了 IP 联动一种**有内容差异化**的设计空间，不是简单素材搬运
- 这给联动叙事**多了一层创作自由度**，跟原作不冲突
