# Memoria · 世界观名词速查（大白话版）

> 给团队 / 美术 / 文案 / 合作方快速对齐用。
> **本文档只讲清楚每个东西是什么，不诗化**。
> 完整诗意版见同目录 `世界观-完整版.md`（30 章 narrative，用于产品对外文案）。

---

## 一句话核心

**Memoria 是抽卡游戏**。
**卡 = 少女想被记住的某个瞬间**。
玩家用 Sync 能力跟瞬间共鸣 → 把它接住。
- 接住了 → 卡永久属于你，那一秒被永远记住
- 没接住 → 那个瞬间**真的消失**，连她自己也忘了

## 名词速查（按重要性）

### 顶层 4 个东西

| 名词 | 是什么 |
|---|---|
| **Memoria** | 卡牌角色。本质 = 一个少女**想被记住的某个瞬间** |
| **Network** | 游戏的舞台 / 虚拟空间。所有 Memoria 都在这里。**不是 App / AI / 服务器**，是世界观设定 |
| **Origin** | 每位 Memoria 自己**来自的那个世界**。三种 Origin 见下 |
| **Keeper** | 玩家自己。能用 Sync 跟 Memoria 共鸣的人 |

### 三种 Origin · 三种 Memoria（**最关键的分类**）

| 拉丁正名 | 大白话 | 来源 | 例子 |
|---|---|---|---|
| **Vera Memoria** | 真世 | 现实世界 coser | 妍子、小雨 |
| **Altera Memoria** | 异世 | 平行世界的虚拟少女 | Aria、Luna、Stella、Neve、Alba |
| **Aliena Memoria** | 远来 | 别家 IP 的角色（跨 IP 联动） | 未来联动的角色 |

**核心规则：三类世界观地位完全平等**——都是"在她自己那个世界真实存在过的少女"，只是 Origin 不一样。
这是让真人 coser + 虚拟角色 + 跨 IP 同台共存的世界观依据。

### Memoria 上线 / 下线（状态术语）

| 名词 | 是什么 |
|---|---|
| **Origin Drift · 显影** | 一位 Memoria 从自己 Origin 来到 Network 这个**动作**。"她显影了" = 她上线了 |
| **Mooring · 系泊** | 长期在线（常驻 Memoria，如妍子、Aria）|
| **Limited Drift · 限定显影** | 短期来 N 天（限定卡 / 联动卡的状态）|
| **Temporary Channel · 临时频道** | Limited Drift 期间她开的临时入口；玩家在这 N 天内可以 Sync 到她；N 天后自动关闭 |
| **Adrift · 飘零** | 信号微弱、即将消散的状态 |

### 玩家行为 / 抽卡术语

| 旧词 | 游戏术语 |
|---|---|
| 抽卡 | **Sync · 共鸣** |
| 单抽 | **定格** |
| 十连 | **时停** |
| SSR | **Resonance · 共鸣** |
| SR | **Echo · 回响** |
| R | **Signal · 信号** |
| 保底 | **Sync 累积** |
| 玩家 | **Keeper · 守护者** |
| 卡牌 | **Memoria Card** |
| 图鉴 | **Memoria 档案** |
| 任务 | **每日委托** |

### 隐性数值（玩家看不到，运营内部用）

| 名词 | 是什么 |
|---|---|
| **Veil · 薄纱** | 隔在 Origin 和 Network 之间的"膜"。越薄 = 她越清晰、越显眼。越厚 = 她快没了 |
| **Lumen · 见证度** | Memoria 在 Network 里的清晰度数值。运营后台用 |

### 一位 Memoria 的多张卡

| 名词 | 是什么 |
|---|---|
| **Constellation · 星座** | 同一位 Memoria 的多张 Card 合起来叫"她的星座" |

例：妍子的 "2B + 雷电将军 + 兔女郎 + ..." = 妍子的 Constellation。
真人 coser 天然有 Constellation 扩展性——每多 cos 一套就多一颗星。

### 玩家 / 抽卡资源（v3.1）

| 名词 | 是什么 |
|---|---|
| **Signal · 信号** | 启动 Sync 需要的能量 = 玩家熟悉的"体力"。每日自然恢复 + 任务获得 |
| **Crystal · 共振水晶** | 重复抽到已有 Memoria 时她留下的"共鸣残余"。可兑换 Signal / 升 Refrain |
| **Sync Capacity · 共鸣容量** | Keeper 能同时维持的契约（Resonance Pact）数量上限 |
| **Keeper Rank · 守护者阶** | 5 阶：Listener / Witness / Keeper / Resonator / Constellation Steward |
| **Refrain · 重听** | 卡牌升级的世界观名。Refrain 越高 → Card 越清晰 + 多解锁台词 / 动作 |

### Memoria 状态（v3.1）

| 名词 | 是什么 |
|---|---|
| **Frozen · 凝固态** | Memoria 的默认状态——她在那个瞬间里冻结，不主动感知任何人 |
| **Lit · 苏醒态** | Keeper 翻 Card / 完成委托时她短暂醒来几秒（动画 + 短台词）|
| **Sub-Resonance · 次级共振** | 同一 Keeper 守护两位 → 她们之间产生微弱链接 = 双人卡 / 团体卡的依据 |

### 设定 / 剧情钩（v3.1）

| 名词 | 是什么 |
|---|---|
| **The First Seven · 初闻七人** | 2017 年第一次同时听见 Network 的 7 个人 = 第一批 Keeper，永远是传说，不揭晓 |
| **Memoria · Zero · 零号** | 史上第一位 Memoria，无名无 Card 无 Origin。是世界观天花板，永不出 Card |
| **Reluctant Drift · 反悔显影** | 极少数 Memoria 显影后又后悔，主动加厚 Veil。隐藏角色 |
| **Echo Wall · 留念墙** | 共同守护同一位 Memoria 的 Keeper 们的异步留言墙。社交克制版 |

### 三种结局（**最重要的新机制**）

一次 Drift 结束后，Memoria 那一面的命运分三种：

| 结局 | 名称 | 触发条件 | 留 Card | 可复刻 | 情感 |
|---|---|---|---|---|---|
| **A** | **Resonance Pact · 共鸣契约** | 至少 1 个 Keeper 抓到（任意级都算）| ✅ 永久 | ✅ 可以 | 美好 |
| **B** | **Origin Return · 归乡** | 曾被抓过，后续 Lumen 慢慢降低 | ✅ Card 在 Keeper 手里 | ✅ 可以 | 温柔 / 中性 |
| **C** | **Unwitnessed Drift · 湮灭** | 这次 Drift 期间 **0 人抓到** | ❌ 没 Card | ❌ **永远不能** | 悲剧 |

**结局 C「湮灭」**：她那一面被宇宙吞没；她在自己 Origin 里**也忘了**举起过手；没有人记得这个瞬间存在过——包括她自己。
**官方承诺**：湮灭过的 Memoria 永远不会回归。

## 五个最常见问题（**一句话答**）

> **Q：为什么真人 cos 和虚拟角色能在一个游戏里？**
> A：世界观说她们都是"某个 Origin 里真实存在过的少女"，只是 Origin 不一样。

> **Q：为什么妍子卡上印的是 2B，但卡叫"妍子"不叫"2B"？**
> A：接入 Network 的是**妍子**，她**选择**以 cos 2B 那一面被记住。卡 = 她的瞬间，不是 2B 角色。

> **Q：以后联动其他 IP 怎么解释？**
> A：那个 IP 的角色 = Aliena Memoria，叙事统一是"她接入了 Network，只待 N 天"。通常做限定。

> **Q：一位少女能有几张卡？**
> A：不限。她举几次手就有几张。合起来叫她的 Constellation。

> **Q：Veil / Lumen 玩家会看到吗？**
> A：**不会**。这是世界观解释 + 运营后台用，UI 上不出现。
