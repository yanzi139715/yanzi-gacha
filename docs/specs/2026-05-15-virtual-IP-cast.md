## 锚点设计原则（v2 准则 · 保留）

✅ **每位必带的真实锚点**：
- 真名（汉语真名 + Network 给的拉丁系英文代号）
- 真实年龄
- 真实职业 / 学生身份
- 来自哪（中国城市级）
- 被记住的**具体日期**（精确到年月日）
- 被记住的**具体地点**（能在 Google Maps 上找到的地标，到县级或景点级）
- 那一刻**具体穿了什么**（户外装 / 通勤装 / 工服 / 便服——生活化）
- 那一刻**具体在做什么、想什么、说了什么**
- 那一刻**身上 / 手里的真实道具**

❌ **明确反对**：
- 水晶冠 / 月牙发饰 / 雪花冠 / 星座王冠
- 纱裙 / 长袖法袍 / 月华杖 / 极光翅膀
- 悬浮 / 发光皮肤 / 瞳孔里有星星
- 「日式审美」或日本本土场景（v3 准则）—— 角色国籍不是日本、不穿和服、不在日本地点
- "cosplay 化"——除非她真的就是 coser（参考妍子）

「**极光 / 月 / 星 / 雪 / 雾」不是她的设定，是那一夜的环境**。
她是个 19 岁学生 / 23 岁摄影师 / 27 岁加班 OL——刚好碰上了那个让宇宙按下快门的瞬间。

## 元设定：一夜的五个时辰

```
  22:47 ─── 23:14 ─── 02:13 ─── 04:53 ─── 05:23
  Aria      Luna      Stella    Neve      Alba
  入夜      子夜      深夜       雪夜静止   拂晓
  极光初起   断电之夜  流星雨    初雪       雾起·夜散
```

五位虚拟 Memoria 不是相互独立的角色，是**同一个不眠之夜的五段时辰**。
不同的人 / 不同的地点 / 同一个时间带 —— 她们的"瞬间"在地球同一晚的不同时区里发生。这条线让 5 位之间有内在叙事张力，也方便后续做"五人海报 / 同夜联动剧情 / 限定活动"。

05:21 她走到平江路中段，河面薄雾从水上升起。
05:23 她突然回头看了一眼——
没什么原因，就是想回头。
那一瞬间薄雾从巷子里漫过来，把她整个人镀了一层柔光。

她对身后空无一人的巷子轻声说：
> "我只来 15 分钟。"
> "够了吧。"

然后转身继续走完剩下的路。

**那一刻她穿什么 / 拿什么**

**揭晓 quote**

**性格底色**
知道自己短暂，所以反而轻盈。
她不想给任何人造成负担——所以她笑得很轻，话也说得很短。
她不是悲伤；她是**释然**。
她比这五位里任何一位都更明白"记住"是什么意思——

**她在等谁**
等一个**不会被她的"短暂"吓退**的人。
不要求你留住她——只要求你在她还在的时候，认真看她一眼。

**限定卡设定**
- **首次出现窗口**：14 天
- **窗口期文案**：
  - 进入窗口期："Alba 出现在了 Network 上——但她说只待 14 天。"
  - 倒数 3 天："薄雾正在散开。"
  - 窗口期结束："她回头说了句『记住我哦』，然后转身走进了雾里。"

## 美术 / 文案使用指南

### Memoria Card 印什么

每张 Card 印 4 行：
1. **Network 代号** · **真名**（如 `Aria · 林夜瑶`）
2. **年龄 / 职业**（如 `23 · 摄影师`）
3. **被记住的时刻**（如 `2024-12-17 22:47 · 挪威 Tromsø`）
4. **揭晓 quote** 作为"她对你说的那句话"

这 4 行的存在让卡牌**有现实质感**，把 Memoria 锚定在"真实存在过"的世界观里。

### 抽卡揭晓动画里用哪句

- Sync 启动："（信号·呼唤）"
- Reveal 揭幕：本档每位的「揭晓 quote」
- 落卡定格：（年龄 · 职业 · 地点 · 日期）四行字幕一层一层浮起来

### 战斗胜利 / 失败台词

留给后续单独 spec。但底线：
- **台词语气基于"她那一秒的状态"**——Aria 安静 / Luna 隐忍 / Stella 数数 / Neve 笑得轻 / Alba 短促释然
- 不能写出"本仙女"、"汝等凡人"、"力量觉醒"这种戏剧化语气——她们是普通女孩
