## 一、资源系统

| 资源 | 范围 | 恢复机制 | 含义 |
|---|---|---|---|
| **心跳**（HP）| 0-15 | 不恢复 | Memoria 的存在度。归零 = 消散 |
| **Sync 能量** | 0-3 | 每回合自动 +1 | 用于"凝固"和"散频"动作 |
| **共鸣值**（蓄能）| 0-10 | 只通过聚频积累 | 用于"爆响"的伤害基础 |

## 二、6 个动作

### 攻击类

#### 触达（普攻）· 快动作
- 消耗：无
- 伤害：基础 2 + 连击递增
- 连击 buff：连续触达第 2/3 次伤害 = 3/4
- 连击归零条件：自己中间出过任何非触达动作，**或**被对方反伤

#### 爆响（强攻）· 慢动作
- 消耗：1 到 N 点共鸣值（玩家自选，N = 当前共鸣值）
- 伤害：1 + 消耗的共鸣值（最高 11）
- **可分配机制**：蓄到 6 点不一定全消耗——可以选 2 点小爆响诱骗对方凝固，留 4 点下回合再用
- 被触达打断：强攻方扣 2 心，共鸣值保留

### 防御类

#### 守护（普防）· 快动作
- 消耗：无
- 效果：挡住触达 + 反伤 1
- **挡不住爆响**（爆响穿透普防）

#### 凝固（有效防）· 慢动作
- 消耗：1 Sync 能量
- 效果：挡住所有攻击（含爆响）+ 反伤 1
- 对空浪费：对方没攻击时，能量也被消耗（举盾的代价）

### 资源类

#### 聚频（蓄能）· 慢动作
- 消耗：无
- 效果：共鸣值 + N，N 按连蓄回合数递增（1 / 3 / 6 / 10，上限 10）
- 连蓄归零条件：自己出过非聚频动作 **或** 被爆响打断

#### 散频（破蓄）· 慢动作
- 消耗：1 Sync 能量
- 效果：**仅当对方本回合也选聚频时**——清零对方共鸣值
- 猜错（对方没聚频）→ 能量浪费 + 这回合无攻无防

## 三、克制基础原则

### 铁律 1：速度决定打断
快动作（触达 / 守护）打断慢动作（爆响 / 凝固 / 聚频 / 散频）。被打断方的动作无效但**资源保留**（蓄能不消耗、能量不消耗）。

### 铁律 2：攻防关系
- 触达 ↔ 守护：攻被反
- 爆响 ↔ 凝固：攻被反 + 凝固消耗能量
- 爆响 ↔ 守护：**爆响穿透**（守护无效）
- 触达 ↔ 凝固：攻被反 + 凝固浪费能量

### 铁律 3：资源动作的代价
聚频和散频本身没有防御——被任何攻击都会受伤。

### 铁律 4：被反伤打断连击
触达被守护/凝固反伤时，自己的连击数清零。

### 铁律 5：聚频被攻击的差异（核心平衡）
- 被触达：扣 2 心，**共鸣值仍进帐**（小骚扰，不能真正打断蓄能节奏）
- 被爆响：扣大血，**共鸣值不进帐 + 连蓄重置**（真正的打断）
- 被散频：清零所有已有共鸣值

## 五、特殊机制详解

### 连击 buff（触达专属）

| 第几次连续触达 | 伤害 |
|---|---|
| 第 1 次 | 2 |
| 第 2 次 | 3 |
| 第 3 次（及之后）| 4（封顶） |

**触发条件**：连续选触达
**归零条件**：自己出过非触达动作 → 立即归零；被守护/凝固反伤 → 立即归零

### 连蓄递增（聚频专属）

| 第几次连续聚频 | 进帐共鸣值 |
|---|---|
| 第 1 次 | +1 |
| 第 2 次 | +3 |
| 第 3 次 | +6 |
| 第 4 次 | +10（上限 10）|

**归零条件**：自己出过非聚频动作 → 立即归零；被爆响打断 → 立即归零；**被触达打中不归零**（小骚扰不能真正打断蓄能节奏）

### 爆响分配机制
- 蓄到 N 点时，选爆响可自选消耗 1 到 N 之间任意值
- 伤害 = 1 + 消耗的点数
- 战术应用：
  - 小爆响（2 点 = 3 伤害）→ 骗对方凝固消耗能量
  - 留下蓄能下回合大爆响 → 对方没能量了挡不住
  - 对方能量为 0 时 → all-in 一击

### 爆响 vs 爆响（差额制）
- 双方都爆响 → 比共鸣值消耗高低
- 高方赢，对方扣**差额伤害**
- 双方共鸣值都按各自消耗扣
- 例：你蓄 6 全爆 vs 对方蓄 3 全爆 → 你 7 vs 对方 4 → 对方扣 3，你不扣，双方共鸣按消耗扣

### 同蓄减半
- 双方都聚频时 → 各自点数减半进帐（向下取整，至少 +1）
- 防止"对蓄消极游戏"——双方都不舒服，逼一方先动手

### 凝固的能量浪费
- 凝固对空（对方没攻击）→ 能量仍消耗 1
- "举盾的代价"——让凝固不能滥用

## 六、胜负判定

- 任一方 **心跳 ≤ 0** → 战斗结束
- 心跳归零方 = 消散（败）
- 另一方 = 共鸣成功（胜）

### 共鸣等级（按胜方剩余心跳分级）

| 胜方剩余心跳 | 共鸣等级 | 叙事 |
|---|---|---|
| > 10 | **Signal · 信号（R）**| 她在远方挥手——稳定但还不到爆点 |
| 3-10 | **Echo · 回响（SR）**| 她朝你伸出手 |
| < 3 | **Resonance · 共鸣（SSR）**| 险胜——她从光中走出，全力回应你 |

**设计哲学**：越险胜共鸣等级越高——羁绊在危险边缘才真正成立。这游戏不奖励"碾压式获胜"，奖励"在快要消散时还能守住"。

```js
{
  id, letter, name,        // 标识 + 视觉
  attackBase,              // 触达基础伤害（默认 2）
  comboMax,                // 连击上限（默认 2）

  guardReflect,            // 凝固反伤（默认 1）

}
```

v3.3 三个神秘人 ABC 数值统一（基础规则演练）。差异化设计后续叠加——每位 Memoria 会有自己的频率个性。

### 三个原型差异化方向（v3.0-3.1 试过的版本，留作参考）

| 角色 | 字段调整 | 玩法定位 |
|---|---|---|
| **A 强袭** | `comboMax: 3` | 连击递增更狠——第 2/3/4 次触达伤害 3/4/5 |

## 八、核心理解（4 句话讲清整个游戏）

1. 这是一场**频率较量**——不是物理伤害，是 Keeper 借给 Memoria 频率对抗 Static 杂讯
2. 核心循环 = **聚集力量（聚频）→ 爆发（爆响）↔ 反制（散频/凝固）↔ 节奏控制（触达/守护）**
3. **速度决定打断**（快攻克慢攻）+ **资源决定深度**（能量 / 共鸣值）= 多层博弈
4. **越险胜共鸣等级越高**——这游戏奖励"在危险边缘守住"的玩家，不奖励"碾压式获胜"
